## 价值
规范的目标是为了帮助我们更好的使用cpp，而不是强制怎么写代码。炫舞一十年前的代码在你看来是怎样的，十年后的人又会以什么样的目光看待我们现在的代码？

遗憾的是本规范目前工具无法支持我们的IDE，所以codereview将是我们的达克摩斯之剑。
